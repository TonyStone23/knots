# knots
Computing the sl3 invarinats of knots and links

## Convention: 

    [contents][knot properties][type].csv

For example:

    [knots][positivealternating].csv 

contains all positive alternating knots.

    [computed][positivealternating].csv 

contains the sl3 generated by the code.

    [knots][positivealternating][truth].csv

contains the ground truth converted frmo HOMFLY.
